# OnlinesaleAi
To run the code on your system
1. Clone the repository .
2. Open a terminal in the directory containing the evaluate_expressions.js file.
3. Run the command npm install node-fetch readline to install the required dependencies.
4. Run the command node evaluate_expressions.js to start the program.
